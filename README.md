قابلیت انتخاب های قابلیت های یک کافه خوب توسط افراد

قابلیت لایک و کامنت و نمره بندی کافه ها توط کاربران

قابلیت دراختیار گرفتن صفحات هر کافه توسط صاحبان آن کافه
